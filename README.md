### Prerequisites

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Atharv1136/KisanDhan.git
   ```

2. Navigate to the project folder:
   ```bash
   cd kisandhan
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

   ```bash
   cp .env.example .env

   ```

   ```bash
   npm run dev
   ```
